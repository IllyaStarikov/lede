# Signal knowledge base

Unpublished working notes, generated by the Signal pipeline. Text and links only — no images, no archive.* URLs.

- `days/` — one deterministic ledger per weekday: every curation completed that day, the day's best uncurated clusters, and ingest activity. Zero LLM involvement.
- `trends.md` — a rolling, model-maintained view of the durable trendlines, updated weekly.

Nothing in this directory is rendered on the site.
